How To Compile


Open a Bash shell. Specify the location of the "apps" folder:

declare -r APPS="/c/res/code/apps"

Type the following commands:

cd $APPS
tools/build.bsh

Multiple executable files and library files result from the compilation:

$APPS/tools/bin/libCommonLib.a
$APPS/tools/bin/Trajectory.exe
$APPS/tools/bin/VarySlantRange.exe
$APPS/tools/bin/VaryHeightOfBurst.exe
    . . .

The Bash script that accomplishes the compilation uses the gcc compiler and the cmake tool:

$APPS/tools/build.bsh

For convenience, modify the $PATH variable in your .bashrc file to include the tools folder:

declare -x PATH=$APPS/tools:${PATH}

For convenience, modify the $env:path variable in your Powershell profile to include the tools folder:

$env:path = "$APPS\tools;" + $env:path

Your Powershell profile is located below your Documents folder:

...\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1
