How To Execute the GENFIT Regression Tests


Open the PowerShell. Specify the location of your local Git working tree:

$WORKING = "$HOME\genfit\working"

Type the following commands:

cd $WORKING
& tools\regression

The PowerShell script that performs the regression testing is:

$WORKING\tools\regression.ps1

The regression test reads each of the input run files contained in the folder:

$WORKING\test\Input

And creates the output XML files in folder:

$WORKING\test\Actual

Then the regression test compares the Actual results with the Expected results in folder:

$WORKING\test\Expected
