How To Execute a Single GENFIT Run


Open the PowerShell. Specify the location of your local Git working tree. And specify the location of the folder containing your run files:

$WORKING = "$HOME\genfit\working"
$RUNS = "$HOME\genfit\runs"

Suppose you have created a run file at:

$RUNS\envrng-01.xml

Type the following commands:

cd $WORKING
& tools\run $RUNS\envrng-01.xml

You will see that in addition to the input envrng-01.xml file, there are now 2 additional output files in the runs folder:

$RUNS\envrng-01-out.xml
$RUNS\envrng-01.csv

The PowerShell script that reads the input file and produces the output files is:

$WORKING\tools\run.ps1

Detailed instructions of how to manually create an input XML run file are at:

User Manual
