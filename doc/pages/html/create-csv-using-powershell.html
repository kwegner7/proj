How To Generate CSV Files of Air Density and Air Depth


Open the PowerShell. Specify the location of the "apps" folder:

$APPS = "C:\res\code\apps"

There is a "jupyter" folder containing a "data" folder. This is a useful place to store the CSV files.

The VarySlantRange command will vary the slant range between burst and receiver and produce a CSV file containing air density and air depth computations. VarySlantRange is the behavior of the ENVRNG run type of GENFIT.

set-location $APPS

VarySlantRange -hob 1000.0 -angle 30.0 -first 1000.0 -last 10000.0 -steps 10
               -file jupyter/data/envrng1.csv

The VarySlantRange command accepts positional arguments as well as named arguments. So the following is equivalent:

set-location $APPS

VarySlantRange 1000.0 30.0 1000.0 10000.0 10 jupyter/data/envrng1.csv

These are other commands available to produce a CSV file of air density and air depth. Each has a -help switch:

VarySlantRange -help
VaryHeightOfBurst -help
VaryAngle -help
Trajectory -help
CsvFileSequence -help
