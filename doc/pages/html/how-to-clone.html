HOW TO CLONE THE GENFIT REPOSITORY


The repository for GENFIT is located at:

\\usqts1nas01\COS1\USCOS1\NavySP\Repo\GENFIT\Code.git

In your work area, decide on a place for the GENFIT working tree. Also, decide on a place to put your own run specifications. These run specifications must be outside of the working tree. For example, place the working tree at:

C:\Users\<your name>\genfit\working

And place the run specifications at:

C:\Users\<your name>\genfit\runs

Open the PowerShell. The value $HOME is the same as "C:\Users\<your name>". Type the following commands (or copy and paste at the PowerShell prompt):

$WORKING = "$HOME\genfit\working"
$RUNS = "$HOME\genfit\runs"
$REPO = "\\usqts1nas01\COS1\USCOS1\NavySP\Repo\GENFIT\Code.git"

cd $HOME
if (-Not (Test-Path $RUNS)) { mkdir $RUNS }
if (-Not (Test-Path $WORKING)) { git clone $REPO $WORKING }
