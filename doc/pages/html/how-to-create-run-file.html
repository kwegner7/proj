How to Create an Input Run File Using the GUI


Open the PowerShell. Specify the location of your local Git working tree:

$WORKING = "$HOME\genfit\working"

Type the following commands:

cd $WORKING\tools\execute
java -jar RES_GUI.jar

When prompted, select the RES.properties file at $WORKING\tools\execute
Set the field Name to "envrng"
Pulldown Classification and select UNCLASSIFIED
Click New

Under the Inputs tab, pulldown Primary Device
Select 001_U_1
Set the field Primary Yield to 10.0
Pulldown Run Type and select ENVRNG
Set the field Height of Burst to 1000.0
Set the field Angle to 30.0
Set the field Minimum Value to 1000.0
Set the field Maximum Value to 10000.0
Set the field Number of Steps to 9

Under the Environment tab, click Add New
When prompted, enter Environment Name, for example, "dose 1" Pulldown Dose Kind and select Dose
Pulldown Response Type and select Total Number Fluence
Check Neutrons under Particles
Click Update

Click Save and name the file, for example, "envrng-test"

There will now exist a new input run file:

$WORKING\tools\execute\envrng-test.xml
