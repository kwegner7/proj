How To Run and View Results at the GUI


Open the PowerShell. Specify the location of your local Git working tree. And specify the location of the folder containing your run files:

$WORKING = "$HOME\genfit\working"
$RUNS = "$HOME\genfit\runs"

Suppose you have created a run file at:

$RUNS\envrng-01.xml

Type the following commands:

cd $WORKING\tools\execute
java -jar RES_GUI.jar

When prompted, select the RES.properties file at $WORKING\tools\execute
Pulldown File and select Import XML
Select the XML file to run, for example, $RUNS\envrng-01.xml

Click Execute
When prompted, enter the output file name, for example, envrng-01

When the execution is complete, you will see that in addition to the input
envrng-01.xml file, there are now 2 additional output files in the runs folder:

$RUNS\envrng-01-out.xml
$RUNS\envrng-01.csv

Then:

Click View Output
  to see the columns of data output from the execution

Click View Graph Output
  Pulldown X to be slant range
  Pulldown Y to be 4pirenv
  Select Make y axis Logrithmic
  Click Plot
