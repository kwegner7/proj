Open the PowerShell. Specify the location of your local Git working tree:

$WORKING = "$HOME\genfit\working"

Type the following commands:

cd $WORKING
& tools\build

There are four files that result from the compilation:

$WORKING\tools\execute\GENFIT.dll
$WORKING\tools\execute\GENFIT.exe
$WORKING\tools\execute\genfit_dose.dll
$WORKING\tools\execute\xerces-c_3_2D.dll

The PowerShell script that accomplishes the compilation is:

$WORKING\tools\build.ps1
